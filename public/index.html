Cara Deploy & Penggunaan

  • Deploy ke Vercel:
    1. Push kode ini ke repository GitHub
    2. Import project di Vercel dari GitHub
    3. Atur environment variable ADMIN_KEY di Vercel dashboard
    4. Deploy dan dapatkan URL live
  • Penggunaan:
    1. Masukkan kunci administrator yang diatur di Vercel
    2. Masukkan kode negara dan nomor telepon target
    3. Klik tombol "Aktifkan TempBan via API"
    4. Sistem akan mengirim 50 permintaan kode registrasi
  • Environment Variables: ADMIN_KEY (kunci rahasia untuk akses API)
