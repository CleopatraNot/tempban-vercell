# TempBan Web - Vercel Deployment

Web interface untuk fitur TempBan WhatsApp yang siap deploy di Vercel.

## Fitur
- Antarmuka web responsif untuk fitur TempBan
- API endpoint untuk proses TempBan
- Autentikasi administrator
- Progress tracking real-time
- Statistik dan logging

## Cara Deploy ke Vercel

### Metode 1: Deploy via Vercel CLI
```bash
# Install Vercel CLI
npm i -g vercel

# Login ke Vercel
vercel login

# Deploy project
vercel
